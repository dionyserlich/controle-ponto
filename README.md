# Controle de Ponto

O objetivo deste projeto é aplicar conhecimentos avançados de Javascript, bem como praticar o uso do ES6 (ES2015) junto com o uso de plataformas BasS (Back-end as a Service).
## Tecnologias

* Javascript (ES6);
* Firebase;
* MaterializeCss;
* SystemJS;
* BabelJS;
